## 3. 可直接上屏的文案（精简版）

### 标题
**研究背景：面向数字制造的几何内核需求**

### 左栏（问题驱动）
- 数字制造流程（建模-分析-加工）对几何算法的稳定性和可复用性要求持续提升。  
- 传统“工具链分散 + 接口割裂”模式下，算法复用成本高，跨语言调用和交互验证效率低。  
- 在孔加工等典型任务中，还存在方向构造、孔深语义、布尔鲁棒性等工程性难点。

### 右栏（项目与切入）
- LibHGP 已形成几何算法基础库，并提供 **C++ / Python / Web** 三层调用路径。  
- 在此基础上，项目进一步面向数字制造场景扩展 CSG 能力，支撑交互式网格编辑。  
- 研究聚焦于：**几何计算能力沉淀 + 跨层接口统一 + 交互结果一致性**。

### 底栏（过渡句）
**因此，本文围绕“可复用几何内核 + 制造任务可落地”展开设计，并以 CSG 打孔任务作为代表性验证场景。**

## 4. 视觉元素建议
